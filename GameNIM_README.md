# === NIM Game – Python Project ===:

## Description:

This project is a Python implementation of the classic **NIM Game**. The game is played with a number of sticks, and each player takes turns removing 1 to 3 sticks. The player who takes the last stick loses.

The project contains two versions:

1. **Simple Version** – Uses basic Python classes.
2. **Advanced Version** – Uses Object-Oriented Programming (OOP) concepts such as inheritance and abstract classes.

## Features:

* Human vs Computer gameplay
* Random game setup
* Input validation
* Computer player with a simple strategy
* Advanced OOP implementation

## Technologies Used:

* Python 3
* Object-Oriented Programming (OOP)
* `random` module
* `abc` module for abstract classes

## How to Run:

1. Download or clone this repository.
2. Open the project in Jupyter Notebook or a Python environment.
3. Run the notebook cells.
4. Follow the instructions displayed in the console.

## Project Structure:

* **Part 1:** Simple version of the NIM game.
* **Part 2:** Advanced version using inheritance and abstraction.
